## 🎉 What's New in v3.2.0

- ✨ **Multi-SSID WiFi** : Automatic connection to multiple WiFi networks
- 🔧 **Separate configuration** : wifi-config.h and config.h files
- 🔒 **Enhanced security** : Sensitive files excluded from Git
- 📌 **Configurable pins** : Centralized hardware configuration
- 🔄 **Auto-reconnection** : Maintains WiFi connection
