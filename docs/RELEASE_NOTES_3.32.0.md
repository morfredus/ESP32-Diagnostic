# Release Notes v3.33.0

**Date de publication : 28 décembre 2025**

## Nouveautés
- Barre de progression TFT fluide
- NeoPixel violet avec effet fondu (Earthbeat)
- Logique BOOT robuste
- Sélection dynamique du contrôleur TFT (ILI9341 ou ST7789) depuis l’interface web
- Synchronisation complète des mappings GPIO/code/documentation

## Améliorations
- Documentation harmonisée, bilingue, versionnée (SEMVER)
- Nettoyage et archivage des anciennes notes de version
- Correction de bugs mineurs et robustesse accrue
